# My Cyber World GPT

An AI-powered cybersecurity assistant built using ChatGPT Custom GPT technology.

![Banner](assets/banner.png)

---

## 🔥 Overview

My Cyber World GPT is a cybersecurity-focused AI assistant designed for:

- Ethical hacking learning
- Cybersecurity education
- Linux troubleshooting
- Networking guidance
- Security awareness
- Penetration testing methodology
- Cybersecurity project assistance

This assistant helps students, researchers, and cybersecurity enthusiasts learn and explore security concepts professionally.

---

## 🚀 Features

- AI-powered cybersecurity guidance
- Ethical hacking learning support
- Linux command assistance
- Networking fundamentals
- Cybersecurity roadmap guidance
- Security awareness education
- Penetration testing methodology
- OWASP Top 10 explanation
- Lab setup support
- Security project ideas
