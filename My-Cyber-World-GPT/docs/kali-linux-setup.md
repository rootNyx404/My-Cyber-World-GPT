# Kali Linux Setup Guide

## Installation Steps

1. Download Kali Linux ISO
2. Create VMware or VirtualBox VM
3. Configure RAM and CPU
4. Install Kali Linux
5. Install VMware Tools
6. Update packages

## Useful Commands

```bash
sudo apt update && sudo apt upgrade -y

```
